\# Drone\_F450 - Drone Forensics Research Project



This project builds an F450 Pixhawk-based GPS drone for digital forensics research.



\## Project Objective



The goal is to build a drone-based investigation lab that can collect and analyze:



\- GPS flight path data

\- Telemetry data

\- Pixhawk flight logs

\- Altitude and battery information

\- Flight mode changes

\- Camera/video evidence in later phases



\## Hardware Platform



\- F450 quadcopter frame

\- Pixhawk 2.4.8 flight controller

\- NEO-M8N GPS with compass

\- FlySky FS-i6X transmitter

\- FS-iA10B receiver

\- 3DR 433MHz telemetry module

\- 3S 11.1V LiPo battery

\- Mission Planner ground station



\## Software Stack



\- Mission Planner

\- ArduPilot / ArduCopter

\- Python 3.10

\- pymavlink

\- pandas

\- folium

\- simplekml

\- matplotlib

\- ExifRead
